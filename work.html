Tolkien Quote Generator

My first forays into web design in the early 2000s were inspired by J.R.R. Tolkien. This is my current homage to this background: a database-powered quote generator with a mobile-first design. using Ruby on Rails, HTML, CSS, and Bootstrap. Uses Git and GitHub for version control, and launched on Heroku.

Korean Adoptee Library

As a Korean adoptee, there are innumerable resources available to help KADs build community, travel to Korea, engage with multimedia, and more. I built this personal project to serve as a resource library for Korean Adoptees - once signed up, users can submit and review resources and bookmark them to curate a personal list of favorite resources.
